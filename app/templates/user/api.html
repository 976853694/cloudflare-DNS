API文档

认证方式

所有API请求需要在Header中携带以下参数：

X-Api-Key: 您的API Key

X-Timestamp: 当前Unix时间戳（秒）

X-Signature: HMAC-SHA256签名

签名算法

message = timestamp + method + path + body

signature = HMAC-SHA256(message, api_secret)
